CSS3

Css3 es utilizado para definir la estética de un sitio web en un documento externo y eso mismo permite que modificando ese documento (la hoja CSS) podamos cambiar la estética entera de un sitio web, el mismo sitio web puede variar totalmente en la apariencia cambiando solo la CSS, sin tocar para nada los documentos HTML o jsp o asp que lo componen. CSS es un lenguaje de estilos empleado para definir la presentación, el formato y la apariencia de un documento de marcaje, sea html, xml, o cualquier otro, la mayor parte de las veces es utilizada para proporcionar dimension visual a archvos html

Beneficios de usar CSS3

  • Reduccion en tiempo de desarrollo y mantenimiento
  • incrementar el rendimiento de las paginas
  • el lenguaje ofrece una amplia gama de herramientas de composición más potentes que HTML
  • e pueden especificar Hojas de Estilo para distintos navegadores

Algunas novedades son:

  • Bordes: bordes redondeados por medio del atributo border- radius
  • Color: colores rgba
  • Fondos: background-origin
  • Texto multi-columna: con esta propiedad el navegador se encarga de producir textos multicolores es decir que maqueta el texto directamente en diversas columnas sin que el programador haga absolutamente nada
  • Fondos: background-origin
