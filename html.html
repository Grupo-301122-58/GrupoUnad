¿Que es el HTML?

Es un lenguaje de programación que es utilizado para el desarrollo y creación de páginas de internet, sus siglas HTML significan “HyperText Markup Language” que es un lenguaje de marcas de hipertexto. HTML nos permite añadir elementos externos a la página como imagen, videos, script, enlaces etc. Esto se puede realizar ingresando la referencia de la ubicación de dicho elemento, utilizando texto y códigos. La web que se crea solo contiene texto y códigos que al momento de visualizarlos en la web nos muestra toso su contenido, este código que se utiliza en el HTML puede ser interpretado por cualquier navegador actual

Breve Historia del HTML

Este lenguaje de desarrollado en el año 1945 por (CERN), con el único propósito de almacenar información y que se pudieran conectar por hipervínculos, luego se diseñó un entorno de trabajo por computadora que poseía un catálogo para facilitar la búsqueda que recibía el nombre de “Online System”. En 1965 Ted Nelson acuño el término hipervínculo lo cual permitió la creación de WWW (World Wide Web) que es un sistema de hipertexto con el cual es posible compartir información de forma variada
